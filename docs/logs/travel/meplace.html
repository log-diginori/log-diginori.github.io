# MEPLACE

  • instagram @meplace.cafe_spaceopen in new window

INFO

MEPLACE는 도시와 공간을 Sustainable Platform으로 만드는 플랫폼 빌더입니다 우리는 도시와 공간에 새로운 기능을 도입하고 유,무형의 자원을 활용하여 경제적, 사회적, 환경적의 재생 및 활성화를 추구합니다. 기존의 기능을 존중하고 삶의 연속성을 확보하여 긴밀하고 새로운 생태계를 만들어 사람들이 모이는 플랫폼을 만듭니다.

MEPLACE는 특정 공간이 본래 가지고 있던 고유기능을 넘어서 사람을 중심으로 지속가능성을 보장하는 것, 그곳에서 협업, 체험, 재생, 개방, 공유,문화를 바탕으로 새로운 정체성을 부여 하고자 합니다.
