  • tensorflow 는 2.x 에서 많은 변화가 있었다. keras 공식 API 를 정식 도입하면서 코드양이 1/10 줄어 들고 쉬워졌다.

  • Flatten 해서 1차원으로 펴서 넣는걸 - 풀리커넥트디 ... 라고 한다.
